Experiência Profissional

Fundação de Amparo à Pesquisa do Estado do Espírito Santo (FAPES - TJES)

10/2022 - o momento

Atualmente atuo no projeto SIGEX do TJES ( Tribunal de Justiça do Espírito Santo ) em pareceria com a FAPES.
Este projeto consiste na criação de um sistema Web para controle interno dos cartórios do Espírito Santo em conjunto com seus acervos e propriedades, no entanto também possui url's públicas.

Atividades:

  • Utilização e configuração do gerador Telosys.
  • Desenvolvimento de API REST/RESTful com Java e Spring Boot.
  • Versionamento de código com Git e GitLab/GitHub.
  • Containerização de aplicações com Docker.
  • Testes automatizados com Rest Assured, JUnit5 e Mockito.
  • Acesso ao banco de dados PostgreSQL e manipulação de dados com Migrations.
  • Documentação de API com Swagger.
  • Implementação de Token de acesso com JWT
  • Desenvolvimento de interfaces responsivas com framework Angular.
  • Programação Reativa com RxJS.
  • Construção de componentes e formularios para aplicação SPA.
  • Configuração de Rotas e Guarda de Rotas no Angular.
  • Testes com Cypress
  • Uso de frameworks CSS como PrimeNG, Bootstrap e Material.
